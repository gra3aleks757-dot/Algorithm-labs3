# Лабораторная работа: Стек и очередь
## Вариативная часть №9 - Запрет одинаковых id

---

## Файлы проекта

**main.py** - классы Task и Queue, демонстрация работы

**tests.py** - тесты для проверки всех функций

---

## main.py

### Класс Task

- `__init__(id, description, priority)` - создает задачу
- `__str__()` - выводит задачу в виде строки

### Класс Queue

Обязательные методы:
- `enqueue(task)` - добавляет задачу в конец очереди + проверка уникальности id (№9)
- `dequeue()` - удаляет задачу из начала очереди
- `front()` - показывает первую задачу без удаления
- `is_empty()` - проверяет пустая ли очередь

Дополнительные методы:
- `size()` - возвращает количество задач
- `clear()` - очищает очередь
- `display_all()` - выводит все задачи на экран
- `find_by_id(id)` - ищет задачу по id
- `remove_by_id(id)` - удаляет задачу по id
- `count_completed()` - считает выполненные задачи
- `has_id(id)` - проверяет есть ли задача с таким id
- `__str__()` - выводит очередь в виде строки

### Функция main()

Запускает демонстрацию работы очереди: добавление, удаление, поиск, вывод на экран.

## tests.py

### Тесты

- `test_enqueue_and_size()` - проверяет добавление задач и размер очереди
- `test_no_duplicate_id()` - проверяет запрет одинаковых id (№9)
- `test_dequeue()` - проверяет удаление из начала очереди
- `test_front()` - проверяет просмотр первой задачи
- `test_is_empty()` - проверяет пустоту очереди
- `test_find_by_id()` - проверяет поиск по id
- `test_remove_by_id()` - проверяет удаление по id
- `test_count_completed()` - проверяет счетчик выполненных задач
- `test_has_id()` - проверяет наличие id
- `test_clear()` - проверяет очистку очереди
- `test_string_representation()` - проверяет строковое представление

### Функция run_all_tests()

Запускает все тесты  и выводит результат.
