## main.py

### Класс Task

- `__init__(id, description, priority)` — создает задачу

### Класс Queue

**Обязательные методы:**

- `enqueue(task)` — добавляет задачу в конец очереди + **проверка уникальности id (№9)**
- `dequeue()` — удаляет задачу из начала очереди
- `front()` — показывает первую задачу без удаления
- `is_empty()` — проверяет пустая ли очередь

## test.py

### Тесты (10 тестов)

1) test_enqueue_and_dequeue — Добавление и извлечение

2) test_front — Просмотр первой задачи

3) test_is_empty — Проверку пустоты

4) test_fifo_order — Порядок FIFO

5) test_dequeue_from_empty_queue — Удаление из пустой очереди

6) test_front_from_empty_queue — Просмотр пустой очереди

7) test_no_duplicate_id — Запрет одинаковых id (№9)

8) test_enqueue_multiple_unique_ids — Добавление уникальных задач

9) test_duplicate_id_does_not_change_queue — Дубликат не меняет очередь

10) test_enqueue_return_value — Возвращаемое значение enqueue

### Функция `run_tests()`

Запускает все тесты и выводит результат.
